# Triple Ads SDK

A lightweight Android library for displaying ads from multiple ad networks through a single API.

## Supported Ad Networks

- ✅ AdMob
- ✅ Google Ad Manager
- ✅ Meta Audience Network
- ✅ AppLovin MAX
- ✅ AppLovin Discovery

---

# Installation

## Step 1. Add the JitPack repository

```gradle
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        mavenCentral()
        maven { url 'https://jitpack.io' }
    }
}
```

## Step 2. Add the dependency

```gradle
dependencies {
    implementation 'com.github.Mr-zombie8145:triple-ads-sdk:1.1'
}
```
